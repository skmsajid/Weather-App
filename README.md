## рҹҡҖ Overview

## вңЁ Features

### рҹҢӨпёҸ Core Capabilities

* **City Search** вҖ” Quickly find weather by city name
* **Live Updates** вҖ” Get real-time weather data
* **Temperature Info** вҖ” Current and feels-like readings
* **Detailed Metrics** вҖ” Humidity, wind speed, and pressure
* **Weather Icons** вҖ” Visual cues for conditions
* **Location Display** вҖ” Shows city and country

### рҹҺЁ User Experience

* **Loading Indicator** вҖ” Visual feedback during data fetch
* **Error Messages** вҖ” Clear and helpful alerts
* **Input Validation** вҖ” Prevents invalid searches
* **Welcome Screen** вҖ” Friendly introduction for users
* **Accessibility** вҖ” Keyboard navigation and screen reader support

### рҹ“ұ Responsive Design

* **Mobile** вҖ” Compact and touch-friendly layout
* **Tablet** вҖ” Balanced spacing and readability
* **Desktop** вҖ” Expanded layout for larger screens

## рҹӣ пёҸ Tech Stack

* **React.js** вҖ” Frontend framework
* **CSS3** вҖ” Styling and layout
* **OpenWeatherMap API** вҖ” Weather data source

## рҹҡҖ Getting Started

```bash
git clone <repository-url>
cd Weather-App
npm install
npm start
```

Create a `.env.local` file:

```env
REACT_APP_WEATHER_API_KEY=your_api_key
```
